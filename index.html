Penerimaan Peserta Didik Baru (PPDB / SPMB)

Silakan lakukan pengisian formulir Dapodik, unggah berkas dokumen fisik, dan pantau hasil seleksi admin secara real-time pada portal di bawah ini.

Status Layanan: Jalur Reguler Terbuka
1. Gunakan NISN Akurat

ID Akun masuk menggunakan NISN resmi siswa. Pastikan tidak salah ketik agar data Dapodik sinkron.

2. Izinkan Akses Kamera

Jika muncul notifikasi izin kamera pada browser browser, pilih "Allow / Izinkan" untuk mempermudah upload foto profil.

3. Cek Status Berkala

Pantau terus dashboard. Catatan revisi alasan ditolak dari operator sekolah akan langsung tampil jika berkas buram.
